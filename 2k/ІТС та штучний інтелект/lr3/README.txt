
Інтелектуальна система моніторингу водного транспорту

Файли:
- db_setup.py — створення БД та початкових записів про судна
- ship_monitor.py — функції для отримання/оновлення даних
- gui_monitor.py — інтерфейс на Tkinter для перегляду суден

Порядок запуску:
1. Запустіть MongoDB сервер
2. Запустіть db_setup.py для створення даних
3. Використовуйте ship_monitor.py або gui_monitor.py для перегляду
