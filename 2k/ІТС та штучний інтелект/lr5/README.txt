
ІНСТРУКЦІЯ ДО ЗАПУСКУ ЛАБОРАТОРНОЇ РОБОТИ №5

1. Переконайтесь, що у вас встановлені Python та бібліотеки:
   pip install simpy matplotlib

2. Запустіть файл:
   python airport_simulation.py

3. Після завершення моделювання буде збережено графік:
   airport_simulation_result.png

Це модель роботи наземних служб аеропорту, яка імітує прибуття літаків,
їх заправку та завантаження, з оцінкою часу обробки.
