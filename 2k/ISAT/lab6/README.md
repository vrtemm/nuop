# Консольна блокчейн-система

## Інструкція запуску

1. Встановіть [Node.js](https://nodejs.org/)
2. У терміналі виконайте:
```bash
npm install
node main.js
```

## Можливості

- Реєстрація (генерація пари ключів)
- Авторизація
- Додавання транзакцій до свого блоку
- Перегляд ланцюга блоків

## Формати файлів

- `blockchain.json` — основний ланцюг
- `private_keys.json` — збережені приватні ключі
